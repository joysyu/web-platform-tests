Visibility collapse

Spec

Collapsing a row group but making its rows visible should still result in a collapsed row group.

row 1 row 2 row 3

Row 1 and Row 2 are both visibility:visible. Row 1 and Row 2 are within the same row group which is visibility: collapse. Rows 1 and 2 should not be visible.
